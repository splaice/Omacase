What is it?

Omacase installs software, configures software, provides themes, and manages a tiling macOS setup. It steals Omarchy's spirit (keyboard-first, one consistent look everywhere, fully reproducible) and translates it to macOS with the compromises that prevents you from fighting macOS.

What's included

Smart default applications with strong default configurations to make them play well together.

window manager

AeroSpace

i3-style tiling — no SIP disabling required. Windows arrange and snap into place; focus, move, and resize them all from the keyboard.

terminal

Ghostty

A fast, GPU-accelerated terminal — the home base your shell, editor, and tools live in.

status bar

SketchyBar

A scriptable bar: workspaces, live CPU & memory, a software-update count, and a keep-awake toggle.

window borders

JankyBorders

A clean border around the active window, so what's focused is always obvious.

keyboard

Karabiner-Elements

Mints a “Super” key from the right ⌘ — the modifier that drives the whole tiling layer.

editor

Neovim + LazyVim

A modern, batteries-included terminal editor, pre-configured and themed to match.

shell

Starship + tmux

A fast prompt and a terminal multiplexer, plus a modern CLI set — eza, bat, fd, ripgrep, fzf, zoxide, atuin.

packages

Homebrew

Every package is declared in one Brewfile and installed with Homebrew — so your whole setup is reproducible on any Mac.

Unified Aesthetics
