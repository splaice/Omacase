Omacase

Opinionated macOS configuration.

Omarchy for Mac. Apple Silicon. Homebrew at /opt/homebrew. Stolen with love, covered in irony.

GitHub Omarchy Themes
Install 
/bin/bash -c "$(curl -fsSL https://omacase.org/install)"

What you get

A tiling Mac desktop, a themed terminal/editor/bar, sane defaults, and a rollback path when you decide you hate it.

  • AeroSpace
  • SketchyBar
  • JankyBorders
  • Ghostty
  • Karabiner-Elements
  • Neovim + LazyVim
  • Starship + tmux
  • Homebrew + Brewfile
  • 21 Omarchy themes

Themes
